# CString

Strings and their functions for C

## Features
- Creating Strings
- Getting the correct length of strings
- String concatenation
- Copying strings

## How to use it?
1. Download str.h from src or github releases
2. Include str.h in your project
```c
#include "str.h"

void fn() {
    str_t* string = createStr(20, "aboba");
}
```
3. Enjoy life
